# 📊 Customer Churn Prediction & Power BI Dashboard

## 🚀 Overview

This project predicts customer churn using Machine Learning and visualizes insights using Power BI. The goal is to identify high-risk customers and support data-driven retention strategies.

---

## 🎯 Objectives

* Predict whether a customer will churn or not
* Generate churn probability scores
* Segment customers into risk categories (High, Medium, Low)
* Build an interactive dashboard for business insights

---

## 🛠️ Tech Stack

* Python (Pandas, NumPy, Scikit-learn)
* Machine Learning (Random Forest)
* Power BI (Dashboard & Visualization)

---

## 📂 Dataset

Used the Telco Customer Churn dataset (IBM sample dataset).

---

## ⚙️ Project Workflow

1. Data Cleaning & Preprocessing
2. Feature Engineering (Tenure, Spending behavior)
3. Categorical Encoding
4. Model Training (Random Forest)
5. Prediction (Churn Probability)
6. Risk Segmentation
7. Power BI Dashboard

---

## 📊 Dashboard Features

* KPI Cards (Total Customers, Churn Rate, High Risk Customers)
* Risk Distribution (Donut Chart)
* Churn Analysis by Tenure & Charges
* Interactive Filters (Contract, Payment Method)

---

## 🧠 Key Insights

* Customers with shorter tenure are more likely to churn
* Higher monthly charges increase churn probability
* High-risk customers can be targeted for retention strategies

---

## 📌 Results

* Generated churn probability for each customer
* Segmented customers into actionable risk groups
* Built an interactive dashboard for decision-making
